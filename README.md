## ✨ Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage calculations (%)
- Decimal number support
- Clear all (AC) button
- Delete last input (DEL) button
- Responsive modern user interface
- Real-time display updates
